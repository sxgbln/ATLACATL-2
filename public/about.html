ATLACATL

ATLACATL es la primera y única red social salvadoreña, creada como proyecto de portafolio para demostrar competencias en desarrollo web full-stack. Esta plataforma integra tecnologías modernas como Node.js, Express.js, MySQL y arquitectura de software escalable, mostrando el dominio de bases de datos relacionales, desarrollo backend y diseño de interfaces responsivas.

Más que una simple demostración técnica, ATLACATL representa un espacio digital que promueve la libertad de expresión y conecta a la comunidad salvadoreña. Con funcionalidades de publicación, comentarios y sistema de likes, esta plataforma establece las bases para el futuro de las redes sociales centroamericanas, combinando innovación tecnológica con valores de comunicación abierta y democrática.
